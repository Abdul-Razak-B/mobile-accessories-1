# mobile-accessories-1
Techo Mobile Accessories — Premium E-Commerce Storefront

Techo is a high-end, single-file e-commerce web application designed with a luxury midnight aesthetic. Engineered for the elite tech market in Pakistan, it combines sophisticated UI/UX with functional e-commerce logic, including a dynamic cart system and a custom WhatsApp-based checkout concierge.

💎 Design Philosophy

The "Architecture of Luxury Tech" is realized through:

Midnight Aesthetic: A deep #050505 palette with royal blue accents and gold-tinted gradients.

Glassmorphism: Navigation and UI elements utilize high-end backdrop-blur filters for a modern, airy feel.

Micro-interactions: Smooth scroll-triggered reveals and hover-state lift effects using CSS cubic-beziers.

Typography: Powered by Plus Jakarta Sans for a professional, tech-forward look.

🚀 Key Features

Elite Preloader: A custom loading sequence to establish branding upon entry.

Dynamic Product Grid: Filterable categories (Audio, Power, Cases) with real-time DOM rendering.

Advanced Cart System: A side-drawer "Shopping Bag" that manages quantities and calculates totals in RS.

WhatsApp Concierge Checkout: A structured order generator that formats customer details (Name, Phone, Address) and cart items into a professional WhatsApp message.

Pexels Integration: High-quality, context-aware imagery for all product listings.

Fully Responsive: Optimized for all devices, featuring a custom mobile navigation overlay.

Concierge FAQ: Interactive, animated FAQ section for improved customer experience.

🛠️ Technical Stack

Structure: HTML5 (Semantic)

Styling: Tailwind CSS

Iconography: FontAwesome 6.4.0

Animations: Animate.css & Native Intersection Observer API

Images: Pexels API/CDN

State Management: Vanilla JavaScript App Object Pattern

📦 Setup & Installation

Since the entire application is contained within a single file for maximum portability:

Download the index.html file.

Open the file in any modern web browser (Chrome, Safari, Edge, or Firefox).

No build step or installation is required.
